# وب‌سایت مؤسسه حقوقی آوای عدالت کریمان

این پروژه وب‌سایت عمومی، فارسی و راست‌به‌چپ مؤسسه حقوقی آوای عدالت کریمان در کرمان با مدیریت آقای علی نظام مجاز است.

## این پروژه با چه فناوری‌هایی ساخته شده است؟

- Next.js برای ساخت صفحات و مسیرها
- TypeScript برای کاهش خطاهای کدنویسی
- Tailwind CSS 3 به‌همراه PostCSS و Autoprefixer برای طراحی واکنش‌گرا و قابل توسعه
- ساختار کامپوننتی برای نگهداری ساده‌تر پروژه

## نسخه‌های پایدار وابستگی‌ها

این پروژه عمداً از نسخه‌های پایدار و سازگار استفاده می‌کند: Next.js 14، React 18، TypeScript 5، Tailwind CSS 3، PostCSS و Autoprefixer.

## اجرای محلی برای مبتدی‌ها

ابتدا وابستگی‌ها را نصب کنید:

```bash
npm install
```

سپس سرور توسعه را اجرا کنید:

```bash
npm run dev
```

آدرس پیش‌نمایش محلی:

```text
http://127.0.0.1:4173/
```

## ساخت نسخه آماده استقرار

```bash
npm run build
npm run start
```

## ساختار اصلی پروژه

- `app/` مسیرها و صفحات سایت را نگه می‌دارد؛ صفحه اصلی، درباره مؤسسه، خدمات، دعاوی ملکی، مقالات و تماس در همین پوشه هستند.
- `components/` اجزای قابل استفاده مجدد مانند هدر، فوتر، فرم تماس و کارت خدمات را نگه می‌دارد.
- `lib/content.ts` متن‌های فارسی و داده‌های سایت را به‌صورت مرکزی نگه می‌دارد.
- `app/globals.css` تنظیمات سراسری Tailwind و رنگ‌های برند را نگه می‌دارد.

## نکته محتوایی

شماره تماس، نشانی دقیق و اتصال فرم تماس در این نسخه به‌صورت قالب آماده باقی مانده‌اند تا فقط اطلاعات رسمی و تأییدشده مؤسسه درج شود.
