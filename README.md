# وب‌سایت موسسه حقوقی آوای عدالت کرمان

وب‌سایت معرفی، خدمات و تماس موسسه حقوقی و داوری «آوای عدالت کرمان». این سایت یک صفحه‌ی
تک‌برگی (single page) و کاملاً واکنش‌گرا (responsive) با پشتیبانی از راست‌به‌چپ (RTL) فارسی است
و بدون نیاز به هیچ فریم‌ورک یا فرآیند بیلدی، تنها با HTML/CSS/JS خالص ساخته شده است.

## ساختار فایل‌ها

| فایل | توضیح |
|------|-------|
| `index.html` | صفحه اصلی سایت (همه بخش‌ها) |
| `styles.css` | تمام استایل‌ها و طراحی واکنش‌گرا |
| `script.js` | منوی موبایل، انیمیشن اسکرول، شمارنده آمار، اعتبارسنجی فرم |
| `404.html` | صفحه خطای ۴۰۴ |
| `assets/favicon.svg` | آیکون سایت |
| `.github/workflows/deploy-pages.yml` | انتشار خودکار روی GitHub Pages |

## بخش‌های سایت

- معرفی (Hero) و آمار موسسه
- درباره ما
- خدمات حقوقی (۸ حوزه تخصصی)
- چرا آوای عدالت؟ (مزیت‌ها)
- فرآیند دریافت خدمات
- معرفی وکلا
- نظرات موکلان
- سوالات متداول (FAQ)
- فرم درخواست مشاوره و اطلاعات تماس

## اجرای محلی

نیازی به نصب چیزی نیست؛ کافی است `index.html` را در مرورگر باز کنید. برای اجرای یک سرور محلی:

```bash
python3 -m http.server 8000
# سپس http://localhost:8000 را باز کنید
```

## انتشار روی GitHub Pages

با هر push به شاخه‌ی اصلی، workflow موجود در `.github/workflows/deploy-pages.yml`
سایت را به‌صورت خودکار منتشر می‌کند.

> **توجه:** انتشار GitHub Pages برای مخازن **خصوصی** نیازمند پلن پولی GitHub است و
> برای مخاطب خارجی قابل دسترسی نیست. برای داشتن لینک عمومی قابل ارسال به همکار،
> یا مخزن را **public** کنید یا از سرویس میزبانی استاتیک دیگری استفاده نمایید.

### فعال‌سازی دستی Pages

`Settings → Pages → Source` را روی **GitHub Actions** قرار دهید (یا حالت
«Deploy from a branch» و انتخاب شاخه‌ی مربوطه و پوشه‌ی `/ (root)`).

## نکات فنی

- فونت: [Vazirmatn](https://github.com/rastikerdar/vazirmatn) از طریق CDN
- بدون وابستگی به کتابخانه خارجی JS
- سازگار با موبایل، رعایت دسترس‌پذیری (a11y) و `prefers-reduced-motion`

## فرم تماس

فرم تماس در حال حاضر صرفاً سمت کلاینت اعتبارسنجی می‌شود و پیام تأیید نمایش می‌دهد.
برای دریافت واقعی پیام‌ها باید به یک سرویس بک‌اند (مثل Formspree، یک API یا ایمیل) متصل شود.
