# وب‌سایت مؤسسه حقوقی آوای عدالت کریمان

یک وب‌سایت عمومی، فارسی و راست‌به‌چپ برای معرفی مؤسسه حقوقی آوای عدالت کریمان در کرمان با مدیریت آقای علی نظام مجاز.

## اجرا و توسعه

```bash
npm install
npm run dev
```

## ساخت نسخه قابل استقرار

```bash
npm run build
```

خروجی نهایی در پوشه `dist` ساخته می‌شود و روی سرویس‌های میزبانی استاتیک قابل انتشار است.

## نکته محتوایی

شماره تماس، نشانی دقیق و اتصال فرم تماس در این نسخه به‌صورت قالب آماده باقی مانده‌اند تا فقط اطلاعات رسمی و تأییدشده مؤسسه درج شود.
