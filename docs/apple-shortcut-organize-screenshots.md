# شورتکات «مرتب‌کردن اسکرین‌شات‌ها» (Apple Shortcuts)

این راهنما قدم‌به‌قدم توضیح می‌دهد چطور در اپلیکیشن **Shortcuts** اپل (روی iPhone / iPad / Mac)
یک شورتکات بسازی که اسکرین‌شات‌های گالری‌ات را به‌صورت مرتب در پوشه‌بندیِ
**سال/ماه** داخل اپ **Files** ذخیره کند و (در صورت تمایل) بعد از مرتب‌سازی، آن‌ها را از
گالری پاک کند تا حافظه آزاد شود.

> **چرا فایل آماده نمی‌گذاریم؟** فایل نهایی شورتکات (`.shortcut`) یک فایلِ امضاشده توسط
> اپل است و فقط روی خودِ دستگاه iOS/macOS قابل ساخت و امضا است؛ بنابراین نمی‌توان آن را
> بیرون از دستگاه تولید کرد. راه درست و پایدار، ساختن آن با همین دستورالعمل است (۵ دقیقه).

---

## کاری که شورتکات انجام می‌دهد

1. همه‌ی اسکرین‌شات‌های گالری را پیدا می‌کند (فیلتر بر اساس نوع رسانه = Screenshot).
2. هر اسکرین‌شات را بر اساس **تاریخ گرفته‌شدن** در مسیر `Screenshots/سال/ماه/` ذخیره می‌کند.
3. (اختیاری) پس از ذخیره، اسکرین‌شات‌ها را از گالری حذف می‌کند.
4. در پایان یک پیام «تعداد X اسکرین‌شات مرتب شد» نشان می‌دهد.

نتیجه چیزی شبیه این می‌شود:

```
Files ▸ On My iPhone ▸ Screenshots
├── 1404/
│   ├── 01-فروردین/
│   ├── 02-اردیبهشت/
│   └── ...
└── 1405/
    └── 03-خرداد/
```

## ساخت شورتکات (قدم‌به‌قدم)

اپ **Shortcuts** را باز کن ← روی **+** (بالا-راست) بزن تا شورتکات جدید ساخته شود ←
اسم آن را بگذار **مرتب‌کردن اسکرین‌شات‌ها**. سپس این اکشن‌ها را به ترتیب اضافه کن
(در نوار جست‌وجوی پایین، نام انگلیسی اکشن را تایپ کن):

### ۱) پیدا کردن اسکرین‌شات‌ها — `Find Photos`
- اکشن **Find Photos** را اضافه کن.
- روی **Add Filter** بزن و این فیلتر را بگذار:
  - **Media Type** `is` **Screenshot**
- (اختیاری) برای اینکه فقط اسکرین‌شات‌های قدیمی‌تر از امروز را جابه‌جا کند، یک فیلتر دوم
  اضافه کن: **Creation Date** `is before` **Today**.
- گزینه **Sort by** را روی **Creation Date** و **Order: Oldest First** بگذار.

### ۲) حلقه روی هر اسکرین‌شات — `Repeat with Each`
- اکشن **Repeat with Each** را اضافه کن و ورودی‌اش را روی خروجی **Find Photos**
  (متغیر **Photos**) بگذار. هر چیزی که بعد از این تا انتهای حلقه اضافه کنی، برای تک‌تک
  عکس‌ها اجرا می‌شود. **Repeat Item** = عکس فعلی.

داخل حلقه، این اکشن‌ها را اضافه کن:

### ۳) گرفتن سالِ عکس — `Format Date`
- اکشن **Format Date** را اضافه کن.
- ورودی **Date** را روی متغیر **Repeat Item** بگذار، بعد روی همان متغیر بزن و
  **Get Details of Photos ▸ Creation Date** را انتخاب کن (تا تاریخِ خودِ اسکرین‌شات گرفته شود،
  نه تاریخ امروز).
- **Date Format: Custom**، رشته‌ی فرمت را `yyyy` بگذار.
- خروجی این اکشن را به یک متغیر با نام **Year** ذخیره کن (اکشن **Set Variable**).

### ۴) گرفتن ماهِ عکس — `Format Date`
- یک **Format Date** دیگر اضافه کن، باز هم با ورودی **Creation Date** همان **Repeat Item**.
- **Custom Format**: `MM-MMMM` (مثلاً می‌شود `03-June`؛ اگر تقویم/زبان دستگاهت فارسی باشد،
  ماه شمسی نمایش داده می‌شود).
- خروجی را در متغیر **Month** ذخیره کن.

### ۵) ساختن مسیر پوشه — `Text`
- اکشن **Text** را اضافه کن و داخل آن این رشته را بنویس (متغیرها را از منوی متغیرها وارد کن):
  ```
  Screenshots/[Year]/[Month]
  ```
  جایی که `[Year]` و `[Month]` همان متغیرهای مرحله‌ی ۳ و ۴ هستند.

### ۶) ذخیره در Files — `Save File`
- اکشن **Save File** را اضافه کن.
- **Input** = **Repeat Item** (عکس فعلی).
- گزینه **Ask Where to Save** را **خاموش** کن.
- **Service/Location** را روی **On My iPhone** (یا **iCloud Drive**) بگذار.
- روی مسیر مقصد بزن و در فیلد **Subpath/Destination Path**، متغیر خروجی **Text** مرحله‌ی ۵
  را بگذار. (اگر پوشه وجود نداشته باشد، خودکار ساخته می‌شود.)
- تیک **Overwrite if file exists** را روشن بگذار تا در اجراهای بعدی خطا ندهد.

### ۷) (اختیاری) حذف از گالری — `Delete Photos`
> اگر **نمی‌خواهی** اسکرین‌شات‌ها از گالری پاک شوند، این مرحله را اضافه نکن.

- **بیرون از حلقه‌ی Repeat** (یعنی بعد از پایان `Repeat with Each`) اکشن **Delete Photos**
  را اضافه کن و ورودی‌اش را روی خروجی **Find Photos** (متغیر **Photos**) بگذار.
  حذف دسته‌ای بهتر از حذف تکی داخل حلقه است (یک‌بار تأیید می‌گیرد).
- iOS یک‌بار اجازه‌ی حذف می‌خواهد؛ عکس‌ها به آلبوم **Recently Deleted** می‌روند و تا ۳۰ روز
  قابل بازیابی‌اند.

### ۸) پیام پایانی — `Show Notification`
- بعد از حلقه، اکشن **Count** را اضافه کن با ورودی **Photos** و خروجی را در **Count** بریز.
- اکشن **Show Notification** با متن: `تعداد [Count] اسکرین‌شات مرتب شد ✅`

## ترتیب نهایی اکشن‌ها (خلاصه)

```
Find Photos  (Media Type is Screenshot, Sort: Creation Date, Oldest First)
Count Photos → Count
Repeat with Each (Photos)
    Format Date (Repeat Item ▸ Creation Date, "yyyy")      → Year
    Format Date (Repeat Item ▸ Creation Date, "MM-MMMM")   → Month
    Text: Screenshots/[Year]/[Month]
    Save File (Repeat Item → On My iPhone, path = Text, Overwrite: On, Ask: Off)
End Repeat
Delete Photos (Photos)        ← اختیاری
Show Notification: تعداد [Count] اسکرین‌شات مرتب شد ✅
```

## اجرای خودکار (Automation) — دلخواه

می‌توانی کاری کنی که این شورتکات خودش هر شب اجرا شود:

1. در اپ Shortcuts به تب **Automation** برو ← **+** ← **Create Personal Automation**.
2. **Time of Day** را انتخاب کن، مثلاً هر شب ساعت ۲۳:۰۰.
3. **Run Shortcut** ← شورتکات **مرتب‌کردن اسکرین‌شات‌ها** را انتخاب کن.
4. **Run Immediately** را روشن کن (در iOS 17 به بعد بدون نیاز به تأیید اجرا می‌شود؛
   ولی اگر مرحله‌ی حذف فعال باشد، iOS برای حذف عکس‌ها تأیید می‌خواهد).

## نکات و عیب‌یابی

- **چرا `Creation Date` و نه تاریخ امروز؟** چون می‌خواهیم پوشه‌بندی بر اساس زمانی باشد که
  اسکرین‌شات گرفته شده، نه زمان اجرای شورتکات.
- **اگر پوشه ساخته نشد:** مطمئن شو در `Save File` گزینه‌ی *Ask Where to Save* خاموش است و
  مسیر در فیلد path قرار گرفته (نه در نام فایل).
- **ماه شمسی می‌خواهی:** زبان/تقویم دستگاه را فارسی کن یا در فرمت از `MMMM` استفاده کن؛
  Shortcuts از تقویم سیستم پیروی می‌کند.
- **سرعت:** برای گالری‌های خیلی بزرگ، فیلتر تاریخ (مثلاً «۳۰ روز اخیر») اجرا را سریع‌تر می‌کند.
- **بازگردانی حذف‌شده‌ها:** عکس‌های حذف‌شده تا ۳۰ روز در آلبوم *Recently Deleted* باقی می‌مانند.
